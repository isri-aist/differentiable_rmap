8 ## Summary
9 This is a ROS package for representing the kinematic reachability of robots: differentiable reachability map.
10 
11 This is a scalar-valued function in task space that is positive only in the region reachable by the robot's end-effector. The main feature is that the scalar-valued function is continuous and differentiable with respect to task-space coordinates. This allows us to formulate the reachability conditions of the robot's end-effectors using reachability maps in continuous optimization for motion planning. The differentiable reachability map is learned using a support vector machine from a sample set of end-effector poses generated from a robot kinematic model.

25 ### Installation procedure
26 It is assumed that ROS is installed.
27 
28 1. Install mc_rtc
29 Installation via apt is recommended. See [here](https://jrl-umi3218.github.io/mc_rtc/tutorials/introduction/installation-guide.html#ubuntu-lts-1804-2004).
30 
31 2. Install jrl-qp
32 ```bash
33 $ git clone https://github.com/jrl-umi3218/jrl-qp -b topic/BlockStructure --recursive
34 $ mkdir build
35 $ cd build
36 $ cmake .. -DBUILD_TESTING=OFF -DBUILD_BENCHMARKS=OFF
37 $ make
38 $ make install
39 ```
40 
41 3. Setup catkin workspace and build
42 ```bash
43 $ mkdir -p ~/ros/ws_differentiable_rmap/src
44 $ cd ~/ros/ws_differentiable_rmap
45 $ wstool init src
46 $ wstool set -t src isri-aist/optmotiongen git@github.com:isri-aist/optmotiongen.git -v ver2 --git -y
47 $ wstool set -t src isri-aist/differentiable_rmap git@github.com:isri-aist/differentiable_rmap.git --git -y
48 $ wstool update -t src
49 
50 $ source /opt/ros/${ROS_DISTRO}/setup.bash
51 $ rosdep install -y -r --from-paths src --ignore-src
52 
53 $ catkin build -DCMAKE_BUILD_TYPE=RelWithDebInfo -DENABLE_JRLQP=ON
54 ```
